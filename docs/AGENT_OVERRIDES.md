# 本项目 Agent 特殊约定

## 1. 项目定位

这是一个 Amazon Seller Central Summary PDF 解析项目，当前已包含：

- 命令行解析脚本
- Flask Web 页面
- 任务状态持久化
- Excel 导出

## 2. 当前业务边界

- 正式统计范围：美国、加拿大
- `MX`、`BR`：明确不纳入当前统计范围，应标记为“跳过”，不要算失败
- 汇率来源：页面维护的 `月份 + 站点` 配置

## 3. 线上环境约定

- 服务器：`8.218.58.28`
- 部署目录：`/opt/summary-amazon-pdf`
- 服务名：`summary-amazon-pdf`
- 端口：`6565`

要求：

- 只允许重启 `summary-amazon-pdf`
- 禁止重启服务器
- 禁止影响同机其他项目

## 4. 开发约定

- 修改需求时，同步更新根目录 `requirements.md`
- 重要阶段结束时，同步更新 `docs/CURRENT_STATE.md`
- 涉及线上发布时，要说明只影响本项目服务
- 若新增站点支持、汇率逻辑、任务统计口径，必须同步更新需求文档

## 5. 交付口径

- 优先给用户“结果”和“可直接使用的页面能力”
- 页面功能要优先考虑真实使用体验：上传反馈、刷新恢复、失败明细、下载链路都要闭环
- 解析逻辑改动后，至少要做一次真实样本验证
